The 7 Phases

Each task follows the same path to mastery

1

Acknowledge

Confirm task understanding. State it back. Remove all ambiguity.

2

Domain Detection

Identify the domain. Map required expertise. Load relevant skills.

3

Check Existing

Search memory. Find prior art. Discover patterns. Check for existing solutions.

4

Research

Deep dive into best practices. 2026 standards. Trade-offs. Alternatives.

5

Decomposition

Break into 10-15 minute chunks. Each with clear deliverables. Dependencies mapped.

6

Synthesize

Plan implementation from research. Architecture decisions. Build order.

7

Execute & Persist

Build each chunk. Test. Commit. Record to memory. Deliver.
