Get IKKF

Get Started with IKKF

Install IKKF in seconds. No dependencies. No complex setup. Just a single binary that works everywhere.

Install IKKF

Choose your platform and run the install command:

# Install via curl
curl -fsSL https://ikkf.info/install.sh | sh

Or download will be available soon.

# Install via curl
curl -fsSL https://ikkf.info/install.sh | sh

Or use your package manager:

# Ubuntu/Debian
apt install ikkf

# Fedora
dnf install ikkf
# Windows version coming soon
# Currently in testing phase

Windows support is coming soon. Join the waitlist to be notified when it's ready.

Initialize IKKF

Create your configuration file and set up the task directory.

$ ikkf init
# Creates ~/.config/ikkf/config.yaml
2

Start Your First Task

Begin with a simple task to see IKKF in action.

$ ikkf start "Build a landing page"
3

Follow the Phases

IKKF will guide you through each of the 7 phases. Answer the prompts and watch your task progress.

$ ikkf status
# Check current task progress
